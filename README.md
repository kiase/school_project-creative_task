#Deep learning

##model_gate
> gate 모델입니다.
> 화면에 음료가 보이는지, main 모델에서 이 음료를 식별할 수 있는지 여부를 판단합니다.

##model_main
> main 모델입니다.
> 화면에 보이는 음료가 어떠한 음료인지 판단합니다.
> 총 7가지 음료가 포함된 2000장의 사진으로 100회씩 학습되었습니다.
